STEP-BY-STEP INSTALLATION (5 MINUTES)

1. OPEN FIREFOX
   - Click the Firefox icon
   - Wait for it to fully load

2. NAVIGATE TO about:debugging
   - Type this in the address bar: about:debugging
   - Press Enter

3. SELECT "THIS FIREFOX"
   - On the left sidebar, click "This Firefox"

4. LOAD THE EXTENSION
   - Click the "Load Temporary Add-on" button
   - Navigate to this folder (SubtitlesArchiver)
   - Select "manifest.json"
   - Click "Open"

5. CONFIRM INSTALLATION
   - You should see "Subtitles Archiver" in the list
   - The extension is now active!

6. PIN TO TOOLBAR (OPTIONAL)
   - Click the extension icon in the toolbar
   - Click the "Pin" icon
   - Icon will stay in toolbar for quick access

FIRST TEST

1. Open any website with videos
   - Try: YouTube.com, Vimeo.com, or any HTML5 video

2. Click the extension icon
   - Look for purple icon in toolbar
   - A popup window will appear

3. Wait for scan to complete
   - You should see "Found X video(s)"

4. Click "üì• Download" next to any video

MAKING IT PERMANENT

The above method creates a TEMPORARY installation (removed on restart).

For permanent installation:

1. Create a .xpi file (packaged extension)
2. Sign it with Mozilla (free developer account)
3. Install from about:addons

OR for personal use:
1. Go to: about:config
2. Search: xpinstall.signatures.required
3. Toggle to: false
4. Then use: about:debugging method above (persists)

TROUBLESHOOTING INSTALLATION

Problem: "Could not load extension"
Solution:
  - Ensure manifest.json is in the correct folder
  - Check file isn't corrupted
  - Try restarting Firefox

Problem: Icon doesn't appear in toolbar
Solution:
  - Refresh the page (Ctrl+R or Cmd+R)
  - Restart Firefox
  - Click ‚â° menu to find extension

Problem: Extension loads but does nothing
Solution:
  - Refresh webpage
  - Check Firefox console (F12)
  - Try on a different website

Problem: "about:debugging" doesn't work
Solution:
  - Ensure Firefox is up to date
  - Try: about:addons instead
  - Restart Firefox

REMOVING THE EXTENSION

Temporary installation removal:
  1. Go to about:debugging
  2. Click "Remove" button next to extension

Permanent uninstall:
  1. Go to about:addons
  2. Find "Subtitles Archiver"
  3. Click the ‚ãØ menu
  4. Select "Remove"
